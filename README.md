# VarDump.css

> Paste messy CSS. Get clean variables.

**VarDump.css** is a single-file CSS variable inspector that extracts, inspects, renames, and exports custom properties from any stylesheet. No install, no build step, no dependencies — just open the HTML file and go.

![VarDump.css Screenshot](screenshot.png)

---

## Features

- **Extract & Inspect** — Paste any CSS and instantly see every `--custom-property`, its value, and how many times it's used
- **Color Swatches** — Color variables get live preview swatches with click-to-copy
- **Bulk Rename** — Click any variable name to rename it; all occurrences update live across the entire stylesheet
- **Unused Detection** — Instantly spot variables that are declared but never referenced
- **Export Clean** — Copy a clean `:root {}` block or the full updated CSS to your clipboard
- **Search & Filter** — Find variables by name or value, filter by used/unused/colors
- **Works Offline** — Zero dependencies, pure vanilla HTML + CSS + JS
- **Single File** — The entire product is one `.html` file

## Why I Built This

I got tired of inheriting projects with 200+ CSS variables scattered across stylesheets. I'd spend hours grepping through files just to figure out which variables were actually used, which were duplicates, and which could be renamed to something sane.

So I built the tool I wished I had — paste in any CSS, get a clean map of every variable with its usage count, rename the messy ones, and export a clean version. All in a single HTML file you can keep in your toolkit forever.

Built in Dar es Salaam, Tanzania.

## How to Use

1. Purchase and download `vardump.html` from Gumroad
